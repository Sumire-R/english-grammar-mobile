# 基本文法ハンドブック - スマートフォン版（全体版）

PDF全17ページの内容を、表紙＋29教材に分けた静的Webサイトです。文章・英語例文・日本語訳はPDF画像のまま掲載しています。APIキー、外部ライブラリ、ビルド作業は不要です。

## ファイル構成

- `index.html`：ページ本体
- `style.css`：スマートフォン対応デザイン
- `script.js`：全教材データ、目次、前後移動、画像拡大
- `assets/`：PDFから切り出した教材画像

## 手元で確認する

`index.html`をダブルクリックするとブラウザで開けます。

## 既存のGitHubリポジトリを更新する

最も分かりやすい方法は、リポジトリの中身をこのフォルダの内容に合わせる方法です。

1. GitHubで、現在公開しているリポジトリを開きます。
2. `index.html`を開き、鉛筆アイコンから内容を置き換えるか、新しい`index.html`をアップロードして上書きします。
3. 同様に`style.css`、`script.js`、`README.md`を上書きします。
4. リポジトリの`assets`フォルダを開き、このフォルダ内の`assets`にある画像をすべてアップロードします。
5. 各操作の最後に「Commit changes」を押します。
6. 数分待って公開URLを再読み込みします。古い表示が残る場合は、ブラウザを再読み込みしてください。

`index.html`、`style.css`、`script.js`は必ず3つとも更新してください。画像は`assets`フォルダの中に置き、ファイル名を変更しないでください。

## 新しくGitHub Pagesを設定する場合

1. リポジトリ上部の「Settings」を開きます。
2. 左側の「Pages」を開きます。
3. Sourceを「Deploy from a branch」にします。
4. Branchを`main`、フォルダを`/(root)`にして「Save」を押します。
5. 数分後、同じ画面に公開URLが表示されます。
